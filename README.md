# 博士学习科研看板（本地版）

一个轻量的单页看板，支持：

- 每日打卡（习惯追踪 + 连续天数）
- 月历与日程标记（组会、实验、投稿 ddl）
- GTD 风格待办（优先级 + 完成状态）
- 科研项目进展（阶段、进度、下一步）

## 如何打开

直接双击 `index.html` 即可在浏览器打开。

如果你希望本地服务方式打开：

```bash
cd /Users/wjsdsg/Personal/20250605-博士学习/cursor
python3 -m http.server 8080
```

然后访问 `http://localhost:8080`。

## 数据说明

- 数据保存在浏览器 `localStorage`（键名：`phd-dashboard-v1`）
- 页面右上角可“重置示例数据”

## 设计参考（借鉴了常见实践）

- **PhD Dashboard 常见模板**：常见是“里程碑 + 文献/任务 + 时间安排”三段式
- **GTD 看板**：任务优先级与状态拆分，减少研究任务模糊性
- **习惯追踪器**：每日打卡与连续天数反馈，用于长期科研节奏维护

这版先是你可直接使用的 MVP。后续可以继续升级：

- 加番茄钟与专注统计自动累计
- 添加“文献数据库”与“论文写作进度”
- 周复盘/月复盘自动生成
- 支持导出 JSON 备份与多端同步
